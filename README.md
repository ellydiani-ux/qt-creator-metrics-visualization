# Plugin para Visualização e Avaliação de métricas em sistemas OO

## Descrição
Projeto e implementação de um plugin para visualização de métricas em sistemas OO, bem como a sua avaliação experimental em atividades de refatoração e evolução
## Problema
Empresas adotam a Programação Orientada a Objetos (POO) para gerenciar a complexidade em grandes projetos de software.
A abordagem OO suporta muitas técnicas para a implementação de sistemas mais flexíveis e de fácil análise, como por exemplo a reusabilidade do código.
Para se ter um maior controle no desenvolvimento dos sistemas e um auxílio no processo de implementação aplica-se as métricas OO.
A IDE Qt Creator possui uma carência de recursos para visualização de sistemas OO. Este recurso é fundamental para a produtividade e qualidade dos projetos.
Com a intenção de suprir essa necessidade foi implementado um plugin para visualização de métricas OO no Qt Creator.

## Solução
A utilização de técnicas para visualização de métricas OO conduz a operações de refatorações mais eficientes e rápidas
A técnica implementada para a visualização de métricas OO foi o plugin Visualization of Metrics.
O principal requisito funcional do sistema é calcular as métricas de cada classe em um projeto OO, exibindo de forma gráfica seus valores e qualificando o software implementado.
